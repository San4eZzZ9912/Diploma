# Autonomous Warehouse Robot System

Дипломный проект по робототехнике.
Система автономного складского робота с компьютерным зрением,
навигацией и backend-сервисом для управления задачами.

## Архитектура
- Робот: Python, ROS, компьютерное зрение
- Backend: Java, Spring Boot, PostgreSQL
- Взаимодействие: REST API

## Структура проекта
- robot/ — код робота
- backend/ — серверная часть
- database/ — структура БД
